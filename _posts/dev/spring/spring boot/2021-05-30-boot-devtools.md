# 스프링 부트 활용

## Spring Boot Devtools

&nbsp;&nbsp;&nbsp;스프링 부트 데브툴즈는 스프링 부트가 제공하는 optional한 툴이다. 즉, 반드시 써야하는 것도 아니고 부트를 사용한다고 기본적으로 적용되는 것도 아니다.
