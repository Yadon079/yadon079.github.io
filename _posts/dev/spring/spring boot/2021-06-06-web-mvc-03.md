# 스프링 부트 활용

## ViewResolve

&nbsp;&nbsp;&nbsp;Spring Boot가 제공하는 ViewResolver 중 ContentNegotiatingViewResolver에 대해서 알아보자.

<b>ContentNegotiatingViewResolver</b>는 들어오는 요청의 Accept Header[^1]에 따라서 응답이 달라진다.
